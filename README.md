# Full Stack Mini Project

## Description
This project is a full stack web application.
The Bus Reservation System Website is a full stack web application designed to simplify the process of booking bus tickets online. It provides a user-friendly interface for passengers to search available buses, view schedules, select seats, and make reservations in a secure and efficient manner.

## Technologies Used
- Frontend: HTML, CSS, JavaScript / React.js
- Backend: Node.js 
- Database: MongoDB

 ## Key Features
 - User registration and login system
 - Search buses by source, destination, and date
 - View bus details and seat availability
 - Online seat booking and reservation
 - Booking history management

  ## 🎯 Objective
The main objective of this project is to provide a convenient, fast, and reliable online bus ticket booking system that enhances user experience and reduces manual booking processes.
